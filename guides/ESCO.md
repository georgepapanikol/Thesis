# Guide: Download and use of ESCO Dataset
## ESCO: What is it?
ESCO is a European classification that includes data on occupations, skills, qualifications, and the relationships between them. This platform provides the ability to download the data in formats suitable for further processing and analysis, both for academic use and for application development.

## Datasets
### Classification
Contains information on occupations, skills, and—most importantly—the relationships between them. These relationships are crucial because they enable the matching of occupations with required skills and can be used in applications such as career guidance, CV–job matching, recommendation systems, and more.

## Installation
* Download URL: [ESCO](https://esco.ec.europa.eu/en/use-esco/download)
* Select: 
    ``` 
    Version: ESCO dataset - v1.2.1
    Content: classification
    Language: en
    File type: csv (49.39 MB)
    ```

## After the download
* Multiple csv files. Some examples:    
    1. `occupations_en.csv`: list with occupations
    2. `skills_en.csv` : skills list 
    3. `occupationSkillRelations_en.csv`: Connection between the skills and the occupations
